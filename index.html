End to End Bengali Text to Speech Synthesis Pipeline

The primary objective of this project, several method has been implemented to recognize Handwritten Bengali Character by a convolutional neural network (CNN). Character is identified to its shapes and several features that help to classification by ResNet34 pre-trained model. This residual network is a deeper neural network that is achieved 3.57% error on the ImageNet data set.
